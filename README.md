# Realflow Inbound AI Agent

An AI-powered inbound voice agent built using **Vapi** and **Cartesia Sonic 3**, designed for **Realflow.co** to qualify inbound commercial real estate leads conversationally.

## Features
- Expressive and empathetic voice (Cartesia Sonic 3)
- Natural conversation flow using Vapi assistant
- Qualification flow: role → asset → location → urgency → contact
- Webhook integration (Node.js or Python)
- Logs structured JSON data locally
- Ready for Supabase or DB extension
